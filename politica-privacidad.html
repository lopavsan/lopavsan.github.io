Política de Privacidad

Esta página web respeta y protege la privacidad de sus usuarios. A continuación, se explica cómo se recopila, utiliza y protege la información, en cumplimiento con el Reglamento General de Protección de Datos (RGPD) de la Unión Europea y la Ley de Privacidad del Consumidor de California (CCPA), así como cualquier otra legislación aplicable.

1. Responsable del Tratamiento de Datos

La administración de esta página web está a cargo de un particular sin ánimo de lucro. No se recopilan datos personales de manera directa; el único objetivo es ofrecer contenido educativo y útil. Para consultas relacionadas con la privacidad, se pueden utilizar los perfiles de LinkedIn o YouTube, cuyos enlaces están disponibles a pie de página.

2. Datos Recopilados y Finalidad

2.1 LocalStorage

Se utiliza LocalStorage para guardar las respuestas y puntuaciones alcanzadas en los exámenes ofrecidos en la plataforma. Estos datos se almacenan exclusivamente en el navegador del usuario y no se comparten con terceros. La información se utiliza únicamente para mejorar la experiencia del usuario, permitiendo guardar el progreso.

Control del usuario: El usuario tiene control total sobre los datos almacenados en LocalStorage. Estos datos pueden eliminarse a través de las opciones del navegador o deshabilitando el uso de LocalStorage en la configuración del mismo. Esto podría afectar negativamente la funcionalidad de la plataforma, como la pérdida de respuestas o puntuaciones alcanzadas guardadas.

2.2 Google Analytics

Se emplea Google Analytics para supervisar el uso del sitio web y mejorar la experiencia del usuario. Google Analytics recoge datos de manera anónima, como la dirección IP, el tipo de dispositivo y el comportamiento de navegación. Los datos recogidos no permiten identificar a usuarios individuales y se utilizan únicamente con fines analíticos.

Más información sobre cómo Google recopila y procesa datos está disponible en su Política de Privacidad.

3. Datos Personales Sensibles

No se solicita ni se recopila información personal sensible (como datos de salud, origen étnico, orientación sexual, creencias religiosas, políticas u otros datos considerados sensibles bajo el RGPD y la CCPA).

4. Cumplimiento con el RGPD y la CCPA

En esta página web no se recopilan ni almacenan datos personales identificables. Se emplean herramientas de terceros, como Google Analytics, para recoger datos anónimos que ayudan a mejorar la experiencia del usuario y el rendimiento del sitio web. A continuación, se explica cómo se gestionan estos datos y los derechos de los usuarios.

Datos Anónimos Recopilados por Terceros

Google Analytics recopila datos anónimos, como la dirección IP, el tipo de dispositivo y el comportamiento de navegación, para fines analíticos. Estos datos no permiten identificar a usuarios individuales y se utilizan exclusivamente para mejorar el sitio web.

Derechos de los Usuarios

De acuerdo con el RGPD, los usuarios residentes en la Unión Europea tienen derecho a:

  • Consultar información general sobre los datos anónimos recopilados a través de Google Analytics. Aunque los datos son anónimos y no se pueden asociar directamente a una identidad específica, es posible solicitar información general sobre cómo se utilizan estos datos.
  • Solicitar que se deje de recopilar datos anónimos en el futuro. Aunque no es posible eliminar datos específicos de usuarios individuales, es posible solicitar que no se recopilen más datos de la navegación contactando directamente. Esto se puede lograr ajustando la configuración de privacidad del navegador.
  • Limitar u oponerse al procesamiento de datos anónimos mediante la configuración de privacidad del navegador. Esta opción permite gestionar cómo se recopilan datos en el futuro sin afectar a otros usuarios.

De acuerdo con la CCPA, los residentes de California tienen derecho a:

  • Conocer qué datos no identificables se recopilan y cómo se utilizan. Para obtener esta información, se puede contactar directamente.
  • Solicitar que se deje de recopilar datos no identificables en el futuro. Aunque no es posible eliminar datos específicos de usuarios individuales, es posible solicitar que se deje de recopilar datos a través de la configuración del navegador.
  • No ser discriminados por ejercer derechos de privacidad.

Para ejercer estos derechos y obtener más información sobre el manejo de datos, se puede contactar a través de los perfiles de redes sociales proporcionados a pie de página. También se puede ajustar la configuración de privacidad del navegador para limitar la recopilación de datos.

5. Uso de Servicios de Terceros

Esta página web incluye enlaces a páginas de terceros, tales como:

  • Schneider Electric (para acceso a información adicional)
  • Google Drive (para la descarga de materiales)
  • GitHub (donde está alojada la página web)

Estos servicios de terceros pueden utilizar cookies y otros métodos de seguimiento. No se tiene control sobre sus políticas de privacidad, por lo que se recomienda revisar las respectivas políticas de privacidad para entender cómo gestionan los datos.

6. Derechos del Usuario

Como usuario de esta página, se tiene el derecho de:

  • Acceder, rectificar o eliminar los datos almacenados localmente en el navegador (como notas o respuestas de exámenes).
  • Eliminar los datos almacenados a través de LocalStorage utilizando las opciones del navegador o deshabilitando el uso de LocalStorage en la configuración del mismo, con el conocimiento de que esto podría afectar negativamente la funcionalidad de la plataforma.
  • Solicitar la eliminación de los datos anónimos recopilados a través de Google Analytics, contactando a través de los perfiles de redes sociales proporcionados a pie de página.

7. Actualizaciones de esta Política

Esta política de privacidad puede actualizarse periódicamente para reflejar cambios en las prácticas o requisitos legales. Se recomienda revisar esta página con frecuencia para estar al tanto de cualquier modificación.
