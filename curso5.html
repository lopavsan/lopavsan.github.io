El Template del Multicarrier

¡Bienvenido a esta formación sobre el template del Multicarrier de PacDrive3! Es un placer acompañarte en este curso, donde aprenderás a dominar una de las soluciones más avanzadas en transporte para la industria: la MultiCarrier MC12 de Schneider Electric. Con este template, integrado en EcoStruxure Machine Expert, tendrás las herramientas necesarias para diseñar, simular y controlar sistemas de transporte de alta precisión y velocidad, facilitando la creación de aplicaciones flexibles y escalables.

¿Qué aprenderás?

A lo largo de este curso, exploraremos juntos el uso del Template del Multicarrier, y adquirirás competencias clave en los siguientes aspectos:

  • Descubrirás cómo configurar gráficamente la forma de tu pista, situar las estaciones de trabajo, añadir carros y diseñar las herramientas que llevarán, todo sin necesidad de escribir código.
  • Explorarás los ejemplos de uso incluidos en el template del multicarrier desde las pantallas de visualización, permitiéndote conocer a fondo las capacidades disponibles.
  • Desarrollarás habilidades para simular tus sistemas de transporte y ajustar tus soluciones en el entorno virtual antes de implementarlas físicamente, ahorrando tiempo y recursos.
  • Aprenderás a crear tu propio modo automático y a aplicar técnicas avanzadas como el warm start (para retomar el modo automático tras una emergencia) y la gestión de zonas seguras (para desactivar carros y segmentos en caso de intrusión).
  • ¡Y mucho más!

Este curso está diseñado para que tengas una experiencia práctica y efectiva. Al final del mismo, tendrás las habilidades necesarias para aprovechar al máximo el potencial de la MultiCarrier MC12 en el desarrollo de aplicaciones industriales modernas y ágiles.

Herramientas necesarias

Para seguir este curso, necesitarás:

  • EcoStruxure Machine Expert

    La plataforma de software de Schneider Electric, necesaria para cargar el template en el controlador. Asegúrate de usar al menos la versión 2.2.0 para garantizar la compatibilidad.

  • Controlador PacDrive3

    El dispositivo en el que cargarás y ejecutarás el template del multicarrier para ver tus aplicaciones en acción. Es necesario, al menos, el modelo LMC402CA10000 para la ejecución del template.

Estoy seguro de que al finalizar el curso, estarás preparado para llevar tus aplicaciones de transporte modular a un nuevo nivel con la MultiCarrier MC12 gracias al template del Multicarrier. ¡Prepárate para comenzar esta emocionante aventura en el control avanzado de sistemas de transporte!
